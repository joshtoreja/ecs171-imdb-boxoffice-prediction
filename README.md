# ECS 171 — IMDb / Box Office Prediction

Team Members: Andrew (Leader), Alexander, Logan

Goal: Predict movie success using metadata and machine learning models.
We will explore classification and regression tasks using public datasets
(IMDb, TMDB, Kaggle).

This repository contains:
- data/: raw and processed datasets
- notebooks/: EDA and experiments
- src/: preprocessing and modeling code
- webapp/: Flask demo app
- paper/: LaTeX files for reports

How to run:
1) Create environment: pip install -r requirements.txt
2) Run baseline notebook in notebooks/
3) Launch demo app from webapp/
